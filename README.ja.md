# Carple Installer

[🇺🇸 English](README.md) | [🇰🇷 한국어](README.ko.md) | 🇯🇵 日本語

Carple Installerは、Carpleのお客様がAAWirelessとAndroid AutoでFermata AutoをインストールするためのAndroidインストールヘルパーです。

このリポジトリは、お客様向けのダウンロードファイル提供のみに使用されます。  
アプリのソースコードは含まれていません。

## ダウンロード

最新のAPKは **Releases** からダウンロードできます。

1. **Releases** を開きます。
2. 一番上の最新バージョンを選択します。
3. `carple-fermata-installer.apk` をダウンロードします。
4. ダウンロードしたファイルを開いてインストールします。

## インストール前の確認

- Android端末が必要です。
- AAWirelessデバイスが必要です。
- 端末でAPKのインストール許可が必要になる場合があります。
- 運転中に動画コンテンツを視聴または操作しないでください。動画機能は、安全で法的に許可されている状況でのみ使用してください。

## お困りですか？

インストールが難しい場合やFermata AutoがAndroid Autoに表示されない場合は、Carpleサポートまでお問い合わせください。
