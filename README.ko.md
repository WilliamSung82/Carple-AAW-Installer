# Carple Installer

[🇺🇸 English](README.md) | 🇰🇷 한국어 | [🇯🇵 日本語](README.ja.md)

Carple Installer는 카플 고객님이 AAWireless와 Android Auto 환경에서 Fermata Auto를 설치할 수 있도록 도와주는 Android 설치 도우미입니다.

이 저장소는 고객 다운로드 파일 제공용으로만 사용됩니다.  
앱 소스코드는 포함되어 있지 않습니다.

## 다운로드

최신 APK는 **Releases**에서 다운로드할 수 있습니다.

1. **Releases**를 누릅니다.
2. 가장 위에 있는 최신 버전을 선택합니다.
3. `carple-fermata-installer.apk` 파일을 다운로드합니다.
4. 다운로드한 파일을 열어 설치합니다.

## 설치 전 확인

- Android 휴대폰이 필요합니다.
- AAWireless 기기가 필요합니다.
- 휴대폰에서 APK 설치 권한을 허용해야 할 수 있습니다.
- 운전 중에는 영상 콘텐츠를 시청하거나 조작하지 마세요. 영상 기능은 안전하고 법적으로 허용되는 상황에서만 사용해 주세요.

## 도움이 필요하신가요?

설치가 어렵거나 Fermata Auto가 Android Auto에 보이지 않으면 카플 고객센터로 문의해 주세요.
