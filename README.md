# Carple Installer

🇺🇸 English | [🇰🇷 한국어](README.ko.md) | [🇯🇵 日本語](README.ja.md)

Carple Installer helps Carple customers install Fermata Auto for use with AAWireless and Android Auto.

This repository provides customer download files only.  
The app source code is not included.

## Download

Download the latest APK from **Releases**.

1. Open **Releases**.
2. Select the latest version.
3. Download `carple-fermata-installer.apk`.
4. Open the downloaded file and install it.

## Before Installing

- An Android phone is required.
- An AAWireless device is required.
- Your phone may ask you to allow APK installation.
- Do not watch or operate video content while driving. Use video features only when it is safe and legal to do so.

## Need Help?

If installation is difficult or Fermata Auto does not appear in Android Auto, please contact Carple support.
